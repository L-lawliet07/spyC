# spyc 

spyc is a tool used to monitor changes in c++ code.  



# Installation

#### Using npm	

> npm install -g spyc



# Usage

To run this tool write

> spyc \<yourfile.cpp> [options]



For CLI options,  use -h or --help

> spyc -h

<img src="./assets/help.gif"/>

##### Example

> spyc ./sample.cpp
